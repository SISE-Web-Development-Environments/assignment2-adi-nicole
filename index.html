In this website we used the jquery Validation Plugin to validate the user inputs from the register form.
We didn't use any templates.
The hardest part of this project was finding bugs. We learned that debugging in JS isn't as simple to execute as
in other programming languages and can become a bit frustrating.
Another part of the project that was hard was making the mosters move properly. They cannot eat anything on the board
and they cannot step over eachother. This part took us the longest to do.
